# Systems of Becoming — Archive Edition

A D3.js force-directed archive tracing relationships among artists, works, concepts, technologies and five generative actions.

## Run

Open the complete folder in VS Code and use **Open with Live Server** on `index.html`.

Do not open `index.html` by double-clicking it, because the browser may block CSV loading.

## Files

- `index.html` — archive interface
- `style.css` — visual system and responsive layout
- `script.js` — D3 network, search, filtering, zoom, drag and record panel
- `data/nodes.csv` — archive records
- `data/edges.csv` — relational links

## Interaction

- Drag nodes
- Scroll to zoom
- Click a node or catalogue entry to inspect it
- Search the catalogue
- Click an action in the top index to filter
- Toggle labels
- Reset the network
