Spatial Canvases

This project explores computational image-making through 2D and 3D canvases. The four works use fractals, underwater environments, glowing geometries, and symbolic organisms to translate mathematical systems into spatial and atmospheric compositions.

Three.js Canvas 1

Golden Mandelbrot Jelly Field

This Three.js canvas translates the Mandelbrot set into a field of glowing parametric spheres. Each sphere is generated from fractal iteration values that control surface color, relief, and perforated openings. Orbit camera controls allow the viewer to navigate the floating geometry in 3D space.

Three.js Canvas 2

Deep-Sea Mandelbrot Tree

This canvas focuses on material, lighting, fog, and atmosphere. A large tree-like structure rises from an underwater rock formation, carrying glowing Mandelbrot spheres like luminous fruit. Dramatic spotlights, emissive materials, floating particles, and flowing fog create a deep-sea spatial environment.

p5.js Canvas 1

Fractal Apparition

This static p5.js canvas uses the Mandelbrot set as a visual structure. A pale fantasy palette, soft grain, gold ribbons, and layered texture transform the mathematical image into an atmospheric fractal apparition.

p5.js Canvas 2

Urchin Pattern Grid

This interactive and animated p5.js canvas turns sea urchins into graphic symbols. Variable grid cells, changing scale, dense outlines, and subtle animation create a patterned spatial surface between ornament, organism, and computation.
